# 🍎 Apple Music Refine (V3)

A premium, modern, and ultra-vibrant skin for YouTube Music inspired by the elegant glassmorphism of Apple Music. This theme transforms your entire interface into a seamless, high-intensity cinematic experience.

---

## ✨ Features

* **🔥  Ambient Dynamic Blur:** Custom-tuned background glow engine that beautifully bleeds album art colors across the entire screen without any performance lag.
* **🎤 Cinematic Animated Lyrics:** Out-of-focus lyric lines are set to a clean 3px blur, while the active line smoothly scales up and snaps into razor-sharp focus with an elegant text-shadow glow.
* **🔍 Liquid Glass Search Suggestions:** Fully refactored search dropdown with strict 8px rounding and frosted transparency, seamlessly blending upper text results and lower card views.
* **📁 Scroll-Proof Frosted Sidebar:** Sidenav elements remain securely locked to a premium transparent frosted glass mesh, completely surviving homepage scrolling states.
* **🎛️ Floating Pill Player Bar:** The stock player bar is reimagined as a floating capsule docking at the bottom, complete with hidden gradients and sleek modern controls.

---

## 🛠️ Installation

1. Make sure you have the **Better Lyrics** extension installed on your browser.
2. Go to the Theme Marketplace inside the extension settings.
3. Search for **Apple Music Refine** (or paste this repository path: `[ankit008-mishra/apple-music-v3](https://github.com/ankit008-mishra/apple-music-v3.git)`).
4. Click **Apply/Install** and enjoy the premium interface!
